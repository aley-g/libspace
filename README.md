LibSpace - Library Seat & Room Booking System
React
Vite
TailwindCSS
Zustand

A robust, full-stack React-based reservation system developed for managing university library study spaces.

🚀 Features

Role-Based Access Control: Distinct experiences for Students, Librarians, and Facility Managers.
Real-Time Room Booking: Check availability and book study rooms, meeting spaces, and computer labs instantly.
Advanced Conflict Resolution: Built-in logic prevents overlapping reservations and respects capacity constraints.
Interactive Dashboard: Visualize library utilization through dynamic charts and metrics using Recharts.
Premium UI/UX: A modern, soft-transition user interface utilizing Tailwind CSS v4 and Lucide icons.

🛠️ Technology Stack

Frontend: React 18, Vite
Styling: Tailwind CSS v4 (PostCSS)
State Management: Zustand (with local persistence)
Routing: React Router v6
Date/Time Logic: date-fns
Notifications: Sonner

💻 Running Locally

To run this project on your local machine:

Clone the repository

Install dependencies:
bash
npm install

Start the development server:
npm run dev

Open http://localhost:5173 in your browser.
